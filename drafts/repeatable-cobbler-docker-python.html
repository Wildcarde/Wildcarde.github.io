Repeatable Cobbler using Docker and Python

Sun 14 February 2021

We've been running a new install of cobbler at work for the last few months that was scratched together in my free time meant to simplify running and maintaining the service. Inspired by our older install managing to corrupt itself so thorougly that we couldn't properly recover it, as well as the install itself being rather cumbersome to maintain. When desinging this setup the goal was an easy to reason about solution that you could spin up as long as you had a few components in the right spaces. It doesn't (currently) do much to hold your hand beyond that but I'm looking to improve it as time permits. The code for this can be found in our teams public work repository.

Base Layer

This is currently built on the centos 7 EPEL variant of cobbler, but work is ongoing to upgrade to the latest stable version of the software. A few API changes have however delayed that for the time being, the EPEL version has proven sufficient for our needs in the mean time.
